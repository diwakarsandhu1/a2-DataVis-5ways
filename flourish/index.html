Flourish

Flourish is a web-based visualization tool focused on rapid, polished storytelling rather than low-level control. Visualizations are created through a graphical interface using pre-built templates, making it easy to produce and share interactive charts without writing code. In this visualization, we can visualize the relationship between penguin flipper length and body mass, with additional encoding for bill length and species using size and color respectively highlighting both the strengths and limitations of a UI-driven approach.
