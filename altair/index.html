Altair

Altair is a declarative Python library for building interactive data visualizations based on Vega-Lite grammar. Instead of manually controlling each visual element, Altair allows you to describe what you want to visualize and alows the library to determine how to render it. This chart below allows us to visualize the relationship between penguin flipper length and body mass, with additional encoding for bill length and species using size and color respectively. Hover over any circle to reveal detailed information such as species, flipper length, body mass, and bill length.
