DataWrapper

DataWrapper is a web-based data visualization tool that allows users to create interactive charts and maps from spreadsheets and other data sources. It provides an easy-to-use interface for designing visualizations without requiring programming knowledge, however due to it being a web-based tool, interactivity options are limited to what the platform provides. In this scatter plot, we visualize the relationship between penguin flipper length and body mass, with additional encoding for bill length and species using size and color respectively. Hover over any circle to reveal detailed information such as species, flipper length, body mass, and bill length.
